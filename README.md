## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion

## Features

- Cinematic parallax scrolling effects
- Scroll-triggered animations
- Glass morphism UI elements
- Grain texture overlay
- Magnetic hover interactions
- Responsive design
- Black & white minimalist aesthetic

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design Philosophy

Brutalist minimalism meets modern AI tooling. Heavy use of negative space, strong typography hierarchy, and motion as a first-class citizen.

## Color System

Strictly black & white:
- Background: #000000
- Surface: #0A0A0A
- Primary text: #FFFFFF
- Secondary text: rgba(255,255,255,0.65)
- Borders: rgba(255,255,255,0.08)
